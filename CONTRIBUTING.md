# Contributing to UrbanPulse AI

First off, thank you for considering contributing to UrbanPulse AI! 

This repository primarily represents my university capstone project and acts as a startup prototype. While I am the sole maintainer, I genuinely appreciate suggestions, feedback, and small contributions from the community. 

## How You Can Contribute

- **Report a Bug**: If something is broken, please open an issue and let me know.
- **Suggest a Feature**: Have an idea to make this better? Open an issue and share it!
- **Submit a Pull Request**: If you'd like to fix a bug or add a small feature, feel free to open a PR. 

## Guidelines

Since this is an academic prototype, the process is straightforward:

1. **Open an Issue First**: Before spending a lot of time writing code, please open an issue so we can discuss it.
2. **Follow Existing Style**: Try to match the existing code style (e.g., standard TypeScript formatting in the frontend). 
3. **Keep it Small**: Small, focused PRs are much easier for me to review than massive overhauls.

## Local Development

If you want to run the project locally to test your changes:

1. Copy `.env.example` to `.env`
2. Run `docker-compose up -d --build` 
3. Run `python seed.py` to populate the database with test data.
